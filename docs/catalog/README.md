# Software Catalog

[← Handbook home](../../README.md)

A categorized, curated index of self-hostable software. Use it to assemble a
short list for a specific need — then read the matching
[concept guide](../../README.md#concept-guides) to understand how the pieces fit
before you install anything.

These lists are broad rather than exhaustive, and they inherit a large body of
links from the original guide. Some will have moved or gone dead over time;
treat the catalog as a well-organized starting point, not a guarantee of
freshness.

## How to use the catalog well

1. **Begin from a need, not a tool.** "I want my photos off the cloud," not
   "I should run twelve services."
2. **Shortlist two or three** candidates and prefer the actively maintained,
   well-documented ones.
3. **Read the relevant concept guide** so you set the service up safely —
   especially **[Security](../security.md)** and
   **[Storage & Backups](../storage-and-backups.md)**.
4. **Run one thing at a time** and get it stable before adding the next.
